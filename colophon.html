Visual Identity

  • Letter Black
    #1b1b1b
    For text.
  • Pale White
    #fbfbfb
    For backgrounds.
  • Full Red
    #ff0000
    For accents only.

Typography

For headlines and titles (like the one above), I use Knockout (in the HTF54 Sumo configuration) with custom kerning. For everything else, the free and open Source Sans Pro in the Light weight.

Website

This website is built with HTML & CSS that I wrote. I use Jekyll as a content management system for flexibility and easy maintenance. The code of rolandomurillo.com is open source and available in GitHub—though I doubt it would be of much use.
